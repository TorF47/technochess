# technochess
## Download
How to download? [Click here](https://github.com/Technochips/technochess/archive/master.zip) to download zip file then rename that to a .love file. Then download [LÃ–VE](https://love2d.org/) and run that .love file with that.
## How to play
It plays like regular chess. Click to select, click to move. Good job! I won't explain how chess work.
* Press S to save the current game into a PGN file. Get your PGN files in `%appdata%\LOVE\technochess`.
* Press any other key to display help.
